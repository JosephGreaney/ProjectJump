To install app on you phone please install CS4084_Project.apk the apk file is included in this zip folder.
The zip should also contain all assets used by the game. To access the scripts of the game, please navigate to the subfolder Assets->ProjectJumpAssets->Scripts
The aim of the game is to get the pickup at the end of the level by navigating through a series of obstacles.
You must then make your way back to the ship at the start of the level before the monster that emerges can catch you.
